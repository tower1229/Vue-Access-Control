## 介绍

Vue-Access-Control是一套基于Vue/Vue-Router/axios 实现的前端用户权限控制解决方案，通过对路由、视图、请求三个层面的控制，使开发者可以实现任意颗粒度的用户权限控制。

整合权限管理功能并做了易用性改进的脚手架可以参考 [CuttingMat](https://github.com/cutting-mat) 项目。
